# Basic Usage

Once installed, you can start interacting with Gemini immediately.

## Opening the Chat Interface

1. Use the Command Palette and search for `Gemini: Start Chat`.
2. Alternatively, configure a shortcut (recommended): `Ctrl+Alt+G` (macOS: `Super+Alt+G`).
3. A new chat view will open with an input area at the bottom.

## Sending Messages

- **Send**: Type your message and press `Ctrl+Enter` (macOS: `Cmd+Enter`).
- **History**: Use the `Up` / `Down` arrow keys to cycle through your previous prompts.

## Real-time Feedback

When Gemini is processing or using tools, you will see visual indicators:
- **Thinking**: The AI is generating a response.
- **Tool Calls**: If the AI needs to perform an action (like reading a file), a permission prompt will appear.

## Reviewing and Applying Changes

If Gemini suggests code modifications, it generates a Diff view. You can review the changes and click the **Apply** or **Discard** buttons within the view to modify your source files safely.
